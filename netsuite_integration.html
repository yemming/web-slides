費用明細自動拋轉 NetSuite 系統

Google Sheets 至 NetSuite 自動化流程

流程核心價值

1. 精準資料篩選

自動從 Google Sheets 篩選出「狀態」為 "Approval" 的費用明細，確保只有經過會計核准的資料才會被處理。

2. 靈活資料映射與轉換

透過中間的資料處理步驟，將 Sheets 中的原始欄位，精準轉換並映射為 NetSuite Expense Report 所需的 JSON 格式，包括員工 ID、子公司 ID、開立日期、類別、金額等。

3. 即時 NetSuite 拋轉

每筆費用明細都會透過 NetSuite API 創建為一筆 Expense Report。確保資料即時且準確地進入企業資源規劃系統。

4. 狀態回寫與錯誤追蹤

根據 NetSuite API 的回應，自動將處理結果 (成功/失敗) 回寫到 Google Sheets，並在失敗時記錄詳細錯誤資訊，大幅提升稽核與錯誤排除效率。成功後也會記錄 NetSuite 連結。

5. 批次處理能力

流程設計為可處理多筆費用明細的批次拋轉，高效應對大量數據同步需求。
